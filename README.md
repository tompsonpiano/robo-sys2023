# robo-sys2023
ロボットシステム学の練習用ディレクトリです。

標準入力によって簡単な四則演算ができる電卓です。

## 必要なソフトウェア
* Python
 * テスト済み： 3.7 ~ 3.10

## テスト環境
* Ubuntu 20.04 LTS
